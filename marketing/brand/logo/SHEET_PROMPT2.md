A clean logo design exploration sheet — a single image, white background, divided into a tidy
4-column x 3-row grid of 12 numbered cells (1–12). Each cell shows ONE distinct, simple,
minimal vector logo mark for a brand called "ember".

BASE CONCEPT (anchor every cell on this — see reference image): a single SIDE-VIEW CHARRED LOG,
drawn as a bold, simple, flat black cylinder lying horizontal, like the attached reference. The
cut END of the log faces the viewer. The 12 variations all riff on THIS log.

THE KEY TWIST (most important): instead of plain spiral tree-rings, the glowing cut END of the
log is a live EMBER that forms a terminal prompt — a ">" chevron or ">_" command prompt /
blinking block cursor — glowing inside the charred ring. The end of the log = a warm glowing
coal = a coding terminal. This fuses "charred log holding heat" with "coding agent." Explore
many ways to render that end: a ">" carved in the ember glow, ">_", a block cursor ▮, a
chevron, concentric rings resolving into a ">", the prompt sitting in a circular coal.

STYLE (critical): flat, simple, monoline / consistent line-weight icon design — a professional
logo designer's black-and-white concept sheet. Solid black marks on pure white; the glowing
ember end may use ONE warm accent (orange) but keep it minimal — mostly black-on-white. NO
gradients beyond a hint, NO 3D, NO photorealism, NO heavy shading, NO drop shadows. Bold,
confident, geometric, reproducible at any size. Each mark iconic, reads instantly.

BRAND: "ember" = an open-source coding agent that keeps running in your own cloud while you step
away — "keep your session warm." A banked charred log holds heat quietly (NOT a flame, NOT a
campfire). Warm, outdoorsy, calm, confident, developer-native.
