The feeling

Close the laptop. It's still burning.

A banked fire doesn't go out when you walk away. Neither does your session. Warm / idle / cold isn't a metaphor — it's the actual mechanic.

warm

Active micro-VM, holding heat. Reopen on any device and it flares back instantly — same workspace, same context, still running.

idle

Coals dimming. Paused but resumable; infra cost drifts toward zero while it waits for you.

cold

Gone out — by choice. Tear it down; the EFS checkout and transcript persist for next time.

Fire a task. Go to the beach bonfire. The build keeps going while you don't.

Enterprise tier

What closes the security review.

SSO & multi-tenant

Cognito or SAML/OIDC federation to Okta, Entra, Google. Real sub stamped on every session, config, and secret.

VPC / PrivateLink

App Runner behind a VPC connector; DynamoDB, S3, Bedrock reached over VPC endpoints. Nothing transits the public internet.

Per-user secrets

Secrets Manager + per-user KMS grants; ABAC on the userId tag. Short-lived scoped GitHub App tokens, not a shared PAT.

Immutable audit

Every turn, shell command, file write, model + token count → write-once store the customer owns. CloudTrail data events on the artifact bucket.

Cost governance

Per-user / per-team budget caps and quotas. Pause or downgrade the model when exceeded.

Admin console

Org dashboard: users, active sessions, spend, audit search. Repo & model allowlists; SSO group → permission mapping.
